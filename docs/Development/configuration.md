# 配置指南

## start_simulation.py 参数配置

编辑 `start_simulation.py` 中的以下参数：

### 基础配置

```python
# 巡飞弹数量（必须与仿真系统设置一致）
UAV_NUM: int = 125

# 损毁比例（0.0 - 1.0）
DAMAGE_RATIO: float = 0.5

# 是否开启随机损毁
IS_RANDOM_DAMAGE = True

# 目标数量（自动计算或手动设置）
TARGET_NUM = int(UAV_NUM / 6)
```

### 通信配置

```python
# Ubuntu控制端IP
UBUNTU_IP = "192.168.1.101"

# Windows仿真端IP
Sim_IP = "192.168.1.102"

# 通信端口
UAV_PORT = 12380
```

### 作战区域配置

默认配置4个矩形区域（可根据实际任务修改）：

```python
RegionList: list = [
    Region(1, [8081, 0, 3752], [8081, 0, 2802], [9519, 0, 2802], [9519, 0, 3752]),
    Region(2, [6007, 0, 2908], [6007, 0, 1840], [7141, 0, 1840], [7141, 0, 2908]),
    Region(3, [7352, 0, 2745], [7352, 0, 1579], [8882, 0, 1579], [8882, 0, 2745]),
    Region(4, [7352, 0, 1532], [7352, 0, 120], [9145, 0, 120], [9145, 0, 1532]),
]
```

### 仿真参数

```python
# ROS通信频率
RATE = 60  # Hz
```

## 仿真系统端配置

1. 点击「仿真参数」
2. 修改「无人机数量」与代码中 `UAV_NUM` 保持一致
3. 仿真区域选择「右上角」
4. 点击「开始仿真」

## 数据记录配置

```python
# 是否开启调试绘图
self.debug = False

# 日志时间戳（自动生成）
self.log_time = datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
```

## 目标类型配置

目标类型定义在 `targets_callback()` 中，可根据任务需求调整：

| 类型值 | 含义 | 所需巡飞弹 |
|--------|------|-----------|
| 0 | 高价值目标 | 4 |
| 1 | 中价值目标 | 3 |
| 2 | 防空目标 | 2 |
| 3 | 低价值目标 | 1 |
| 4-7 | 其他类型 | 2-3 |
